devbox VM management

Host setup (run once, before creating the VM):
  make host-setup        Run all host setup steps in sequence
  make install-libvirt   Install libvirt/KVM toolchain for govirt local provider
  make install-nfs       Install nfs-kernel-server
  make setup-paths       Install /etc/host-shadow-paths
  make install-export-gen Install /usr/local/bin/shadow-export-gen
  make export-nfs        Run shadow-export-gen to (re)generate /etc/exports
  make enable-libvirt    Enable and start libvirtd and the default network
  make enable-nfs        Enable and start nfs-kernel-server
  make ufw-allow         Allow NFS through ufw (if ufw is active)

VM lifecycle:
  make create-vm         govirt create devbox
  make delete-vm         govirt destroy devbox
  make ssh-vm            govirt ssh devbox
  make status-vm         govirt status devbox
